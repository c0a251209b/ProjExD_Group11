# ヴェルダラ：砕かれた門

## 実行環境の必要条件
* Python >= 3.10
* pygame >= 2.1
* 外部画像・外部音声ファイルは使用していないため，追加の素材ダウンロードは不要

## ゲームの概要
* プレイヤーは，壊れたポータルによって危機に陥った島「ミラ」を救う探索者である。
* 砂浜や遺跡を探索し，シャドウスライムを倒して経験値，コイン，クラフト素材を集める。
* レベルアップによりステータスを上昇させ，スキルを強化し，回復アイテムや矢をクラフトしながら冒険を進める2D見下ろし型アクションRPGである。
* 参考URL：なし（世界観，ゲームシステム，GUIはオリジナルで制作）

## ゲームの遊び方
* `W`，`A`，`S`，`D`キーまたは矢印キーでプレイヤーを移動する
* `SPACE`キーで近接攻撃を行う
* 経験値が一定量に達するとレベルアップし，HP，MP，Attack，Magic Attack，Speedが上昇する
* レベルアップ時にはSkill Pointを1獲得する
* `I`キーでインベントリ画面を開く
* `K`キーでスキル画面を開く
* `T`キーでステータス画面を開く
* メニュー表示中は，画面上部のタブをクリックして画面を切り替えることもできる
* `ESC`キーでメニューを閉じる。通常画面で`ESC`キーを押すとゲームを終了する
* HPが0になるとゲームオーバーになる

## ゲームの実装
### 共通基本機能
* Pygameによるゲームウィンドウとゲームループ
* 見下ろし型マップの描画
* プレイヤーの移動と画面内での位置制限
* 敵の追跡，攻撃，当たり判定
* HP，MP，EXP，レベルアップの管理
* 基本GUI（HPバー，MPバー，EXPバー，ミニマップ，クエスト表示，ホットバー）
* `os.chdir(os.path.dirname(os.path.abspath(__file__)))` により，実行場所に依存しないファイルパス処理

### 分担追加機能
* インベントリ機能（担当：学籍番号）：所持アイテムと素材を一覧表示するGUI
* スキル機能（担当：学籍番号）：Skill Pointを消費してAttack，Magic Attack，HP，Speedを強化するGUI
* ステータス機能（担当：学籍番号）：Level，HP，MP，Attack，Magic Attack，Speed，EXPを確認するGUI
* 戦闘機能（担当：学籍番号）：近接攻撃，Fire Orb，敵へのダメージ，敵の撃破処理
* マップ・世界観デザイン（担当：学籍番号）：

### メモ
* 外部素材に依存せず，地形，キャラクター，敵，アイテム，GUIはPygameの図形描画を中心に実装している
* プレイヤーの主要ステータスは，Attack，HP，MP，Speed，Magic Attackである
* 敵を倒すとランダムでWood，Herb，Crystal Dustを入手できる
* Small HP Potion：Herb ×2，Crystal Dust ×1
* スキルは各レベル最大3まで強化できる
* Sword Mastery：Attack +3
* Arcane Focus：Magic Attack +3
* Vitality：Max HP +15
* Swift Step：Speed +15
